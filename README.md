# BiliBiliLiveStreamFetcher

用于抓取哔哩哔哩直播间的直播流。傻瓜化操作，怎么用都写在程序里了。

无需安装，可直接运行；自带 MPV 封装，规避 Referer 和 User-Agent 限制。

注意：部分直播间直播流有 1 小时过期时间，1 小时后需要重新获取直播流，此程序暂未实现更新直播流功能。

## 使用方法
通过 GitHub Release 下载最新版，在 Windows 10 (1511) 以上版本运行即可。

此程序需要 .NET Framework 4.6.1 才能运行，Windows 10 (1511) 以上版本已自带该框架，可以直接运行。更低版本的操作系统需要手动安装 .NET Framework 4.6.1。

## 构建
建议使用 Visual Studio 2022 打开项目进行构建。

需要依赖 .NET Framework 4.6.1 目标包。
